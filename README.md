# Cursor Portfolio Project

## Objective

This repository documents the setup process completed as part of the 100Hires portfolio assignment.

## Tools Installed

- Cursor IDE
- Claude Code Extension
- OpenAI Codex Extension
- Git
- GitHub

## Steps Completed

1. Installed Cursor IDE.
2. Installed the Claude Code extension.
3. Installed the OpenAI Codex extension.
4. Created a public GitHub repository.
5. Installed Git.
6. Cloned the GitHub repository into Cursor.
7. Created this README file.
8. Committed the README and pushed it to GitHub.

### Claude Code authentication
